LIST

UL

The HTML <ul> tag defines an unordered (bulleted) list. Use of this tag together with the <li> tag to create unordered lists.

[EXAMPLE]

  1. ElementA
  2. ElementB
  3. ElementC

OL

The HTML <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical. The <li> tag is used to define each list item.

[EXAMPLE]

  • ElementA
  • ElementB
  • ElementC
